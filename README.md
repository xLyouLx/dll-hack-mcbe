# Minecraft Bedrock Edition Cheating Clients (MCBE / Win10/11) ENG

Collection of popular & old Horion-based cheat clients for versions **1.12 – 1.21.13+**

### Top list (very subjective 2025–2026)

| Client     | Game versions          | Main purpose                         | Key features / notes                                           | AC bypass power |
|------------|------------------------|--------------------------------------|----------------------------------------------------------------|-----------------|
| Horion     | 1.12 – 1.21.101        | Casual play + weak servers           | Most stable & lightweight classic                              | ★★☆☆☆           |
| Borion     | <1.20 – 1.21.13X+      | PvP, HVH, fun                        | Horion fork, many extra modules (TP Aura, Phase, etc)          | ★★☆☆☆           |
| Solstice   | 1.21+                  | Beautiful UI + PvP                   | Very nice GUI, has some weak bypasses                          | ★★★☆☆           |
| Solstice forks | 1.21+              | Same + improvements                  | Many forks are more stable & stronger than original            | ★★★–★★★★        |
| Dorion     | 1.20 (sometimes 1.20.X)| Pure HVH / HVP                       | Open-source Horion fork, focused on fights, often crashes      | ★★★★☆           |
| Zephyr     | 1.19.XX                | Trolling + HVH                       | Tons of features incl. playing Minecraft sounds to everyone    | ★★☆☆☆           |

### Folder contents

- **clients/** — the .dll files themselves  
  File name almost always includes supported game version
- **configs/** — ready HVH configs for roughly half of the clients  
  just drop them into the client's config folder

### Quick summary

- **Horion** — best choice if you just want to play and survive longer than 3 minutes  
- **Borion** — Horion but with way more modules  
- **Solstice** — if you want good-looking menu + some actual bypass attempts  
- **Dorion** — when you're going full sweat HVH and can tolerate crashes  
- **Zephyr** — trolling server with sounds + decent PvP when configured right
